# 快速操作指南

## 退出程序

### 方法一：正常退出（推荐）
在程序等待输入时，输入：
```
quit
```

### 方法二：强制退出
如果程序卡住或没有响应，按：
```
Ctrl + C
```

### 方法三：查找并终止进程
如果程序在后台运行，使用以下命令：
```bash
# 查找进程
ps aux | grep "python.*main.py" | grep -v grep

# 终止进程（替换 PID 为实际进程号）
kill <PID>

# 或者强制终止
kill -9 <PID>
```

### 停止所有相关进程
```bash
pkill -f "python.*main.py"
```

### 检查程序是否在运行
```bash
ps aux | grep "python.*main.py" | grep -v grep
```
