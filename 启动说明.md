# 如何启动程序

## 方式一：测试模式（推荐先使用这个）

在终端中运行：
```bash
cd /Users/zhouyinyin/Downloads/booksearch
python3 main.py --test
```

**界面说明：**
1. **终端界面**：会显示提示信息，你可以输入书名进行测试
2. **投影窗口**：会弹出一个全屏的黑色窗口，用于显示高亮效果

**操作步骤：**
1. 运行命令后，会看到终端提示："请输入书名:"
2. 输入书名，例如："rethinking users" 或 "coffee lids"
3. 投影窗口会显示红色高亮框
4. 输入 "quit" 退出程序
5. 按 'q' 键关闭投影窗口

## 方式二：语音识别模式

在终端中运行：
```bash
cd /Users/zhouyinyin/Downloads/booksearch
python3 main.py
```

**界面说明：**
1. **终端界面**：显示系统状态和识别结果
2. **投影窗口**：全屏显示，用于高亮书籍
3. **麦克风**：系统会自动监听你的语音输入

**操作步骤：**
1. 运行命令后，系统会校准麦克风（保持安静）
2. 说出书名，例如："rethinking users"
3. 系统会自动识别并高亮显示
4. 按 Ctrl+C 退出程序
5. 按 'q' 键关闭投影窗口

## 注意事项

- 投影窗口是全屏的，可能会覆盖其他窗口
- 如果看不到投影窗口，检查是否有其他全屏应用
- 首次使用需要授予麦克风权限（macOS系统设置中）
- 书籍位置坐标需要根据实际书架调整（编辑 book_database.py）
